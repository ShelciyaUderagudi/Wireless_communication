# Wireless Channel Throughput Simulation

This project simulates wireless channel conditions using a Gaussian-distributed 
SNR model to analyse throughput behaviour across multiple channel realizations.

## What it does
- Simulates 1000 random wireless channel conditions using Gaussian shadowing
- Computes Shannon capacity for each channel realization
- Converts dB SNR values to linear scale for accurate capacity calculation
- Plots throughput distribution as a histogram for statistical analysis

## Tools & Libraries
- Python 3
- NumPy
- Matplotlib

## Background
Shannon capacity defines the theoretical maximum data rate of a communication 
channel. This simulation models realistic channel variability through random 
SNR generation, providing insight into average and worst-case throughput 
in wireless systems.
